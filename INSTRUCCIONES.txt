  App completa. Incluye estructura, estilos verde limon, union de PDFs y generacion de correctivos.

- .nojekyll
  Archivo recomendado para GitHub Pages.


COMO SUBIRLO

Sube todo el contenido de esta carpeta a la raiz de un repositorio de GitHub.
Despues activa GitHub Pages desde Settings > Pages.


IMPORTANTE

No borres index.html. Es el archivo que GitHub Pages abre como pagina principal.

La descarga principal del informe es Word. Tambien hay descarga en PDF.

Los PDFs, criterios y modelo quedan guardados en el navegador hasta pulsar eliminar.
